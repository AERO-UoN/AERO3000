# AERO3000 — Flight Dynamics: Tutorial Code

Teaching MATLAB scripts for the AERO3000 Flight Dynamics course.

## Course Progression

| Week | File(s) | Topics |
|------|---------|--------|
| 1 | `tut1_1.m`, `tut1_2.m`, `tut1_3.m` | MATLAB basics, functions, velocity computation, altitude profiles |
| 2 | `tut2_1.m`, `tut2_2.m` | Coordinate frames, 3-2-1 rotation matrices, Euler angles, quaternions |
| 3 | `tut3_1.m`, `tut3_2.m` | Curve fitting, lateral force equilibrium, roll angle from sideslip |
| 4 | `tut4.m` | Longitudinal static stability, neutral point, static margin |
| 5 | `load_stol_data.m`, `tut5.m` | Lateral-directional equilibrium, control authority, damaged aircraft analysis |
| 6 | `EOM_Example.m` | Introduction to Newton's equations of motion |

## Reference Textbooks

- Nelson, R. C. (1998). *Flight Stability and Automatic Control* (2nd ed.). McGraw-Hill.
- Stevens, B. L., Lewis, F. L., & Johnson, E. N. (2016). *Aircraft Control and Simulation* (3rd ed.). Wiley.
- Stengel, R. F. (2004). *Flight Dynamics*. Princeton University Press.

## Requirements

- MATLAB R2018b or later (no additional toolboxes required)

## Usage

Open each weekly folder and run the tutorial scripts in order. Each script is self-contained.
